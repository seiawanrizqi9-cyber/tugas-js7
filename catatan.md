# Catatan

## Penggunaan Method Array JavaScript

```javascript
//1
namaTeman[] // Untuk mencari isi(nama teman) di dalam array
namaTeman.lenght // Untuk mencari panjang array yang ada

//2
.push // Untuk menambahkan elemen baru ke dalam array

//3
.pop    // Untuk menghapus elemen terakhir dari array

//4
i ++ // Untuk increment/menambahkan nilai i

//5
.indexOf // Untuk mencari indeks dari elemen tertentu dalam array (true/false)

//7
.unshift // Untuk menambahkan elemen baru di awal array

//8
.shift // Untuk menghapus elemen pertama dari isi dalam array

//9
.concat // Untuk menggabungkan dua atau lebih array menjadi satu array

//10
.reverse  // Untuk membalik urutan elemen dalam array

//11
.sort // Untuk mengurutkan elemen yang ada didalam array

//12
.includes // Untuk mencari apakah elemen tertentu ada dalam array (true/false)

// 13
.slice // Untuk mengambil bagian tertentu dari dalam array

//14
.forEach // Untuk melakukan perulangan pada setiap elemen dalam array

//15
.map // Untuk melakukan perulangan pada setiap elemen dalam array dan diubah menjadi hasil array baru

//16
.filter // Untuk melakukan perulangan pada setiap elemen dalam array dan mengambil hasil sesuai kondisi tertentu

//17
.reduce // Untuk melakukan perulangan pada setiap elemen dalam array dan menghitung nilai yang sesuai dengan kondisi tertentu

//18
.find // Untuk mencari elemen dalam array yang sesuai dengan kondisi tertentu
.startWith() // Untuk mengecek apakah string dimulai dengan huruf/angka tertentu

//19
.some // Untuk mengecek apakah ada elemen dalam array yang memenuhi kondisi tertentu
.every // Untuk mengecek apakah semua elemen dalam array memenuhi kondisi tertentu

```
